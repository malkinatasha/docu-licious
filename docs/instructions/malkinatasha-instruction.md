# Создание имени пользователя (username)

Имя пользователя — это ваш публичный адрес в Telegram в формате `@ваше_имя`. Имя пользователя позволяет вас найти в Telegram без номера телефона. Кроме того, создавая имя пользователя, вы получаете личную ссылку: `t.me/ваше_имя`. Перейдя по ней, человек сразу откроет диалог с вами в Telegram.
Данная инструкция описывает, как создать имя пользователя в мобильном приложении на Android.

## Порядок действий:

1. В приложении Telegram перейдите в меню **Настройки**.

2. Выберите пункт меню **Учетная запись**.

3. Выберите пункт меню **Имя пользователя**. 

4. Введите желаемое имя латиницей:

    * Используйте только латинские буквы.

    * Можно использовать цифры и нижнее подчеркивание `_`. 

    * Нельзя использовать кириллицу и пробелы. 

>Важно: Имя должно быть уникальным во всем Telegram. Если оно занято, вы увидите сообщение «Это имя пользователя уже используется».

5. Нажмите галочку в правом верхнем углу (или кнопку **Готово** на клавиатуре), чтобы подтвердить выбор.
