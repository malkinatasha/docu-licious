# Документационный портал Docu-licious 

## Добро пожаловать в Docu-licious
Здесь вы найдете документацию по сервисам, используемым в нашей компании.

## Чем Docu-licious отличается от корпоративной базы знаний?
Документационный портал Docu-licious дополняет корпоративную базу знаний и служит основным хранилищем для технической документации по нашим сервисам.
